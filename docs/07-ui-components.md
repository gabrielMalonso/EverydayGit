## Objetivo
Garantir consistência visual e simplicidade de manutenção da UI.

## Tipografia
- Fonte do sistema.
- Hierarquia clara: títulos, labels, texto secundário.

## Componentes base
- **Button** (primário, secundário, ghost)
- **Input** (com prefixos/ícones)
- **Panel** (cards com borda e background elevado)
- **List Item** (ícone + label + status)
- **Badge** (status, branch atual)

## Layout principal
- Top bar fixa.
- Painéis em colunas (branches, changes, AI).
- History no rodapé com scroll independente.

## Acessibilidade
- Contraste alto.
- Foco visível.
- Labels de ícones.
